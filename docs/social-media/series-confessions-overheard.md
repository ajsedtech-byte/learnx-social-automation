- Confessions: Plain text on white/off-white background. Notes-app font (SF Mono or system default). Small LearnX logo, bottom-right, low opacity. No design. That's the point.
- Overheard: Chat-bubble format (iMessage or WhatsApp style). Each speaker gets a different bubble color. Narrator text in italic below. Small LearnX branding at bottom.

**Posting cadence:**
- Student Confessions: Every Sunday (Instagram Story + Feed post)

**Engagement strategy:**
- End each post with "Tag someone who does this" or "Type 'ME' if this is you"
- Use carousel format for longer conversations (Overheard series)
- Encourage UGC: "Send us YOUR school confession for a chance to be featured (anonymously)"

**Hashtags:**
- Confessions: #StudentConfessions #LearnX #HarBacchaStar #SchoolLife #IndianStudents #Relatable

- Shares per post > 500 (these formats are built for sharing)

**LearnX feature integration ratio:**
